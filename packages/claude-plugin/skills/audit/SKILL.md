# I-Evolve: Audit

Show the audit trail for memory governance decisions.

Steps:
- List recent actions: `i-evolve audit list`.
- Show all actions for a memory: `i-evolve audit show <memory_id>`.
- Explain the policy decision: `i-evolve audit explain <memory_id>`.
