# I-Evolve: Explain Memory

Help the user understand a memory.

Steps:
1. Fetch the memory: `i-evolve memory get <id>`.
2. Show its decision history: `i-evolve audit explain <id>`.
3. Summarize: what it is, why it is scoped this way, and when it expires.
